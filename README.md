### About

I build tools that run locally, stay private, and actually work. Currently deep in AI agent orchestration, macOS native tooling, and full-stack TypeScript.

- **StanleyOS** — local-first AI portfolio tracker: Gmail → PDF parser → Ollama analyst → Slack alerts, no data leaves the machine
- **AI Pulse** — macOS menu bar app that tracks Claude Code and Codex CLI token usage in real time
- **Orchestr8** — CLI that runs a PRD through a multi-agent PM → architect → dev pipeline
- **Claude Code Analytics** — personal usage dashboard for the Claude Code CLI
